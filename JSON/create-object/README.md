# create-object

Given *unlimited* number of named params, returns valid JSON.

```html
<cms:call 'create-object' name='Vagabond' type='Yacht' is_cat='1' has_sail='YES' />
```
Results in -

```json
{"name":"Vagabond","type":"Yacht","is_cat":"1","has_sail":"YES"}
```


## Parameters
* named, &ndash; any number
