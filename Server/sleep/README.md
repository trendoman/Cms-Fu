# sleep

Make server wait. Count in seconds.

```xml
<cms:call 'sleep' for='0.500' />
```

```xml
<cms:call 'sleep' '5' />
```

**NOTE: This functionality may be disabled by shared hosting provider â˜º (use VPS instead).** Check phpinfo on server to review the directive ***disable_functions*** under ***Core***.


## Parameters

* for &mdash; time in seconds

## Usage

Add 'sleeping' code that imitates a heavy page loading for a while (*5* seconds)

```xml
<cms:call 'sleep' '5' />
```

Internally func employs PHP function *usleep*.

## Installation

```xml
<cms:embed 'funcs/Server/sleep/sleep.func' />
```
